TRANSLATION QUALITY STANDARDS
================================================================================

✓ Modern Standard Arabic (MSA)
  - Proper diacritical marks (tašdīd, fatḥah, ḍammah, sukūn, tanwīn, etc.)
  - Formal academic tone
  - Grammatically correct Arabic syntax
  - Appropriate for educational context

✓ Professional French
  - Pedagogical tone matching existing translations
  - Consistent terminology with lesson vocabulary
  - Clear, accessible explanations
  - Academic but not overly technical

✓ Format Preservation
  - All newlines (\n) maintained
  - Transliteration formatting preserved
  - Code escaping maintained
  - Special characters unchanged

✓ Consistency
  - Terminology aligns with vocabulary lists in each lesson
  - Titles match their content explanations
  - Same level of detail as existing translations
  - Consistent with lesson objectives
